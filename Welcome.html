Acerca de greenbsys

Cuenta una página web moderna desarrollada en PHP con el motor de base de datos Mysql, la misma que incorpora un servidor web con apache, servidor de base de datos, servidor sftp; incorpora la opción para subir archivos y un servicio de correo electrónico.

Recomendaciones:

.Realiza tus registros de clientes y consultas de manera segura

.No de a personas desconocidas sus datos de acceso

.Se recomienda subir archivos al sistema máximo de 2 MB
